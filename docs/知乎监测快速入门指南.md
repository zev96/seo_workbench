# 知乎监测快速入门指南（v2.2）

## 🚀 5分钟快速上手

---

## 第一步：启动程序

```bash
python main.py
```

点击左侧导航栏的 **"🔍 知乎监测"**

---

## 第二步：配置品牌词库（首次使用）

1. 点击顶部工具栏的 **🏷️ 品牌管理** 按钮
2. 添加我方品牌：
   - 品牌名称：`CEWEY`
   - 类型：选择"我方品牌"
   - 点击"添加品牌"
3. 添加竞品品牌（可选）：
   - 品牌名称：`戴森`、`石头`等
   - 类型：选择"竞品品牌"
4. 关闭品牌管理对话框

---

## 第三步：配置Cookie（强烈推荐）

### 为什么要配置Cookie？
- ✅ 提高检测成功率（>95%）
- ✅ 某些回答需要登录才能查看
- ✅ 降低被封风险

### 如何获取Cookie？
1. 打开浏览器，登录知乎
2. 按 `F12` 打开开发者工具
3. 切换到 **Network** 标签
4. 刷新页面
5. 点击任意请求
6. 在 **Headers** 中找到 `Cookie`
7. 复制完整的Cookie字符串

### 配置步骤
1. 点击 **⚙️ 设置** 按钮
2. 粘贴Cookie到输入框
3. **勾选"启用温和模式"**（推荐）
4. 点击"保存设置"

---

## 第四步：添加监控任务

### 方式1：单个添加
1. 点击 **➕ 添加监控**
2. 输入知乎问题链接
   - 示例：`https://www.zhihu.com/question/123456789`
3. 选择监测品牌：`CEWEY`
4. 检测范围：`20`（默认）
5. 点击"添加"

### 方式2：批量导入（推荐）
1. 点击 **⬇️ 下载模板**
2. 保存Excel模板
3. 打开Excel，填写多个问题链接：

| 问题链接 | 目标品牌 | 检测范围 |
|---------|---------|---------|
| https://www.zhihu.com/question/123456789 | CEWEY | 20 |
| https://www.zhihu.com/question/987654321 | CEWEY | 20 |
| https://www.zhihu.com/question/111222333 | CEWEY | 15 |

4. 保存Excel
5. 点击 **📥 导入Excel**
6. 选择刚才保存的Excel文件
7. 自动导入完成

---

## 第五步：执行检测

### 选择性检测（新功能）
1. **勾选**要检测的任务（表格第一列的复选框）
2. 或点击表头的 **☐** 全选所有任务
3. 点击 **🔄 立即检测**
4. 等待检测完成（进度条显示）

### 查看结果
检测完成后，表格会显示：
- **✅ 在榜**：找到了品牌（绿色背景）
- **❌ 未发现**：未找到品牌（红色背景）
- **⚠️ 失败**：检测失败（黄色背景）

排名列会显示：
- `第 2, 5, 8 名`（找到的具体排名）
- `-`（未找到）

## 第七步：设置定时任务（可选）

1. 点击任务列表中的 **⏰** 按钮
2. 勾选"启用定时任务"
3. 设置执行时间：`10:00`（每天早上10点）
4. 点击"保存"
5. ⏰ 按钮变为绿色，表示已启用

---

## 第八步：导出报告

1. 点击 **⬇️ 导出报告**
2. 选择保存位置
3. 自动生成Excel文件，包含：
   - 问题标题和链接
   - 目标品牌
   - 当前状态和排名
   - 浏览量/关注者
   - 定时任务配置
   - 最后更新时间

## ⚠️ 常见问题速查

### Q1：检测失败怎么办？
1. 检查Cookie是否有效
2. 启用温和模式
3. 增加请求间隔
4. 等待10分钟后重试

### Q2：为什么只检测了部分任务？
- 确认是否勾选了要检测的任务
- 点击表头的☐可以全选

### Q3：Top10数据不完整？
- 启用温和模式可能只获取部分数据
- 关闭温和模式可以获取更多数据
- 点击详情页的"刷新数据"重新抓取

### Q4：遇到知乎异常弹窗？
1. 立即停止检测
2. 启用温和模式
3. 更新Cookie
4. 等待1小时后重试

## 💡 最佳实践

### 推荐配置
- ✅ 启用温和模式
- ✅ 设置有效Cookie
- ✅ 请求间隔：8-15秒
- ✅ 定时任务：凌晨2点或早上10点
- ✅ 监控数量：≤20个

### 使用技巧
1. **批量导入**：一次性添加所有监控问题
2. **选择性检测**：只检测重要的问题
3. **定时任务**：设置自动检测，解放双手
4. **定期清理**：删除不需要的监控
5. **导出报告**：定期保存数据
